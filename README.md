# Mood to Poem

Next.js App Router 기반의 한국어 감성 문장 생성 앱입니다.

## 시작하기

```bash
npm install
npm run dev
```

환경 변수:

```bash
OPENAI_API_KEY=your_key_here
```

## Vercel 배포 시 환경 변수 설정

- `OPENAI_API_KEY`는 **Vercel Production Environment**에 반드시 설정되어야 합니다.
- 환경 변수 추가/수정 후에는 **반드시 재배포(redeploy)** 해야 런타임에 반영됩니다.
